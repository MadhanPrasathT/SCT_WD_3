# Interactive Quiz Game

An interactive and user-friendly **Quiz Game Web Application** built using **HTML, CSS, and JavaScript**.  
This application presents different types of questions, collects user answers, and displays the final score at the end.

---

## Features

- Displays quiz questions dynamically
- Supports multiple question types:
  - Single choice (Radio buttons)
  - Multiple choice (Checkboxes)
  - Fill in the blanks
- Validates user answers
- Calculates and displays final score
- Simple and responsive user interface
- Beginner-friendly JavaScript logic

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## Question Types Supported

1. **Single Select Questions**
   - User selects one correct option.

2. **Multi Select Questions**
   - User selects multiple correct options.

3. **Fill in the Blanks**
   - User types the correct answer in a text field.

---

## How to Run the Application

1. Download or clone the project.
2. Save the file as `index.html`.
3. Open the file in any modern web browser.
4. Start answering the quiz questions.
5. View your final score at the end.

---

## How the Application Works

- Questions are stored in a JavaScript array.
- Each question is loaded dynamically.
- User input is validated based on question type.
- Score is updated after each question.
- Final score is displayed once all questions are completed.
